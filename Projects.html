Projects
Below are some of my favorite projects I've worked on in the past few years.
Splendor
Splendor is a card game where players compete to build Renaissance Europe's most lucrative jewelry business by acquiring gems, purchasing mines, and gaining the favor of nobles. This implementation of Splendor was used a Java backend and a JavaFX frontend. We used technologies such as REST, Docker, Maven, and Git in developing it as part of our year-long project for COMP 361 at McGill University. Feel free to try it out for yourself here.
Ray Tracer
One of my favorite courses in university has been Fundamentals of Computer Graphics. Learning about all the processes that take place to bring a coded image onto a screen has transformed the way I think about TV, movies, and games. This ray tracer project is an implementation of the technology that has become so prevalent in graphics today. Feel free to explore some sample images and the code behind them here.
Sample Operating System
This project was a sample operating system coded in C. A simple implementation of some of the key components that modern operating systems employ. It included features such as demand paging, multithreaded task scheduling, and more. You can explore the code behind it here.
Light at the End
During the early months of the pandemic a group of friends that I met while at the National High School Game Academy at Carnegie Mellon University organized a game jam. In a team of three, Light at the End was our creation. A witty point and click adventure game where a player must make decisions, trade resources, and travel through a post apocalyptic New York City Subway system. We used Unity game engine to build it, check it out here.
